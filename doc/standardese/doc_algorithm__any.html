template <typename F = identity>
using any = find_if<F, always<bool_<true>>, always<bool_<false>>>;

resolves to true_ if any element in the input pack fulfills the provided

predicate, otherwise false_. \requires Type F shall be a continuation predicate and C shall be any continuation. example call<any<same_as<void>>,void,int,char> resolves to true_.
