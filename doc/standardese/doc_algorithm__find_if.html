Class template kvasir::mpl::find_if

template <typename F = identity, typename FC = listify, typename NFC = FC>
struct find_if
{
    template <typename ... Ts>
    using f = typename detail::find_if_impl<detail::first_find(sizeof...(Ts)), FC, NFC>::template f<F::template f, Ts...>;
};

finds the first element in a list for which a predicate holds

Effects: calls FC with all elements in the input pack except those which occure before the first element for which the provided predicate F holds. if F doe not hold for any elements NFC is called instead of FC \requires Type F shall be a continuation predicate and FC and NFC shall be any continuation. \example call<find_if<same_as<int>,cfe<std::tuple>>,void,int,char> resolves to std::tuple<int,char>.
